 
# Employee Management Application

## Overview
A mobile application developed using Flutter and Firebase to manage employee records efficiently in an organization.

## Features
- Secure login using Firebase Authentication
- Role-based access (Admin & Employee)
- Add, update, and manage employee details
- Cloud Firestore for real-time data storage
- Clean and responsive user interface

## Tech Stack
- Flutter
- Dart
- Firebase Authentication
- Firebase Firestore

## Learning Outcomes
- Mobile application development using Flutter
- Backend integration with Firebase
- CRUD operations and role-based access control
- Real-world application design and problem-solving
